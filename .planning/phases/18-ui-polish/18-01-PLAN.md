    `components/ui/animated-pressable.tsx`:
    - Create `AnimatedPressable` via `Animated.createAnimatedComponent(require('react-native').Pressable)`.
    - Export `ScalePressable` component accepting `PressableProps` plus optional `scaleValue` (default 0.96).
    - Use `useSharedValue(1)` for scale, `useAnimatedStyle` for transform.
    - `onPressIn`: `scale.value = withSpring(scaleValue, { damping: 15, stiffness: 300 })`, also call forwarded `onPressIn`.
    - `onPressOut`: `scale.value = withSpring(1, { damping: 15, stiffness: 300 })`, also call forwarded `onPressOut`.
    - Style merge: `[style, animatedStyle]` with animatedStyle LAST to avoid transform override (Pitfall 3 from RESEARCH.md).
    - IMPORTANT: Base styles of cards that use ScalePressable must NOT set `transform` -- this is already true in current codebase.

    `components/ui/animated-heart.tsx`:
    - Accept `{ isBookmarked, onToggle, size?, color }` props.
    - Use `useSharedValue(1)` for scale.
    - `handlePress`: if `!isBookmarked`, play pop animation via `withSequence(withSpring(1.4, { damping: 8, stiffness: 400 }), withSpring(1, { damping: 12, stiffness: 200 }))` and call `Haptics.impactAsync(Haptics.ImpactFeedbackStyle.Light)`. Always call `onToggle()`.
    - Render: `Pressable` > `Animated.View` with animatedStyle > `MaterialCommunityIcons` name=heart/heart-outline.
    - `hitSlop={8}` on the Pressable.
    - Size defaults to 20.
