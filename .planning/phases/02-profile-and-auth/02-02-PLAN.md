    **1. Supabase project**
    - Go to https://supabase.com/dashboard and create a new project (or open existing)
    - Copy Project URL and anon key from Settings -> API
    - Open TheCook/TheCook/.env.local and fill in:

    - Go to https://console.cloud.google.com (create a project if needed)
    - Create a Web OAuth 2.0 client ID and an iOS OAuth 2.0 client ID
    - Set iOS client Bundle ID to match ios.bundleIdentifier in app.json
    - Fill in .env.local:
      EXPO_PUBLIC_GOOGLE_WEB_CLIENT_ID=your-web-client-id.apps.googleusercontent.com
      EXPO_PUBLIC_GOOGLE_IOS_CLIENT_ID=your-ios-client-id.apps.googleusercontent.com
    - Add reversed iOS client ID as URL scheme in app.json under ios.infoPlist.CFBundleURLTypes:
      { "CFBundleURLSchemes": ["com.googleusercontent.apps.YOUR_IOS_CLIENT_ID"] }

    - Apple Developer Portal -> Identifiers: enable "Sign In with Apple" for your App ID

    - Generate a private key (.p8) with Sign In with Apple capability
    - Configure Apple provider in Supabase Dashboard -> Authentication -> Providers -> Apple
