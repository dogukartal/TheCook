**File naming:** kebab-case, no Turkish characters in filename.
  Good: `karniyarik.yaml`, `baklava.yaml`, `ezme-salata.yaml`
  Bad: `karnıyarık.yaml`, `Baklava.yaml`

**Minimum required fields (top-level):**
```yaml
id: kebab-case-no-turkish-chars
title: Türkçe başlık
cuisine: türk
category: ana yemek   # one of: ana yemek | kahvaltı | çorba | tatlı | salata | aperatif
mealType: dinner      # one of: breakfast | lunch | dinner | snack | dessert
skillLevel: beginner  # one of: beginner | intermediate | advanced
prepTime: 15          # minutes, integer
cookTime: 30          # minutes, integer
servings: 4           # integer
coverImage: null      # ALWAYS null for v1
allergens:
  - gluten            # any of: gluten | dairy | egg | nuts | shellfish | sesame | soy | fish | mustard | celery | sulphites | lupin | molluscs
equipment:
  - fırın             # any of: fırın | blender | döküm tava | tencere | tava | bıçak | rende | karıştırıcı | zırh | buharlı tencere
ingredientGroups:
  - label: null       # null for single-group recipes, string for multi-group (e.g. "Hamur için")
    items:
      - name: Soğan
        amount: 1
        unit: adet    # any of: adet | gr | kg | ml | litre | su bardağı | çay bardağı | yemek kaşığı | tatlı kaşığı | tutam | demet | dilim | diş | paket | kutu | avuç | dal
        optional: false
steps:
  - instruction: Adımı açıklayan cümle.   # if contains colon, use >- block scalar
    why: Bu adımın neden önemli olduğu.
    looksLikeWhenDone: Doğru yapıldığında nasıl görünmeli.
    commonMistake: En sık yapılan hata.
    recovery: Hata olduğunda ne yapmalı.
    stepImage: null   # ALWAYS null for v1 — write explicitly, do not omit
    timerSeconds: null  # null if no timer; integer seconds if timed (e.g. 300 = 5 min)
```

**Instruction with colon — requires block scalar:**
```yaml
    instruction: >-
      Malzemeleri ekleyin: un, yağ ve tuz.
```
