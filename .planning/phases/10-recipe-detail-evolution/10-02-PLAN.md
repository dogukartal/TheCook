**RED phase:** Fill in the test.todo stubs from Plan 01 in __tests__/recipe-adaptation.test.ts with real test implementations. Import the pure functions from useRecipeAdaptation.ts (which doesn't exist yet, so tests will fail on import).

Test data: Create a mock recipe with 2 ingredient groups:
- Group 1: Tereyagi (1 yemek kasigi, scalable=true, alternatives=[{name:"Zeytinyagi", amount:1, unit:"yemek kasigi"}]), Tuz (1 tutam, scalable=false, alternatives=[])
- Group 2: Yumurta (4 adet, scalable=true, alternatives=[])

**GREEN phase:** Create TheCook/src/hooks/useRecipeAdaptation.ts with:

1. Export pure function `formatAmount(amount: number): string` -- show integer for whole numbers, 1 decimal otherwise, strip trailing zero.

2. Export pure function `scaleIngredientGroups(groups: IngredientGroup[], multiplier: number): IngredientGroup[]` -- deep map: for each ingredient, if scalable !== false, multiply amount by multiplier and round via Math.round(val * 10) / 10. Return new arrays (immutable).

3. Export pure function `applySwaps(groups: IngredientGroup[], swaps: Record<string, string>): IngredientGroup[]` -- for each ingredient whose name is a key in swaps, find the matching alternative in ingredient.alternatives by comparing against swap value, replace name and amount with alternative's values. If no match found in alternatives, leave unchanged.

4. Export pure function `resolveStepVariables(instruction: string, ingredientMap: Map<string, {name: string; amount: number; unit: string}>): string` -- regex replace `{{(\w+)\.(name|amount|unit)}}` using ingredientMap lookup. For amount, use formatAmount. Unmatched patterns left as-is.

5. Export hook `useRecipeAdaptation(recipe: Recipe | null)` that:
   - useState for servings (init from recipe.servings)
   - useState for swaps: Record<string, string> (init empty {})
   - Compute multiplier = servings / recipe.servings
   - useMemo to derive adaptedGroups via scaleIngredientGroups then applySwaps
   - useMemo to build ingredientMap from adaptedGroups (Map of ingredient name -> {name, amount, unit})
   - useMemo to derive adaptedSteps: map recipe.steps with resolveStepVariables on each instruction
   - Export: { servings, setServings, swaps, swapIngredient(name, subName), resetSwap(name), resetAll(), multiplier, adaptedGroups, adaptedSteps, ingredientMap }
   - swapIngredient sets swaps[name] = subName
   - resetSwap removes name from swaps
   - resetAll resets servings to recipe.servings and swaps to {}

Run tests. All should pass.
