Chip row: horizontal ScrollView (showsHorizontalScrollIndicator=false), contains Chip components in order: "Hepsi" (selected when selectedCategory===null), "Kahvaltı", "Çorba", "Ana Yemek", "Tatlı", "Salata", "Aperatif" (mapped from CategoryEnum). Each chip calls onCategoryChange with the corresponding CategoryEnum value or null for "Hepsi". Chip labels map Turkish display names: ana yemek→'Ana Yemek', kahvaltı→'Kahvaltı', çorba→'Çorba', tatlı→'Tatlı', salata→'Salata', aperatif→'Aperatif'. Import Chip from @/components/ui/chip.

Advanced filter toggle: small Pressable with ▼ arrow text (rotate 180° when open using Animated.Value or simple state toggle with transform style) at the end of the chip row. When open, renders an advanced filter panel below the chip row as a View (no modal) containing three sections:
  1. Cuisine: "Türk" / "Dünya" toggle chips → updates filter.cuisine
  2. Cook time: "< 15 dk" / "15–30 dk" / "30+ dk" chips → updates filter.cookTimeBucket ('under15'/'15to30'/'over30')
  3. Skill: "Başlangıç" / "Orta" / "İleri" chips → updates filter.skillLevel

Each advanced chip calls onFilterChange with the updated DiscoveryFilter. If user taps an already-selected chip, deselect it (set to null). Panel has a light border, borderRadius=12, padding=16, backgroundColor='#F9FAFB', marginTop=8.
