State:
- `allIngredients: string[]` (loaded once on mount from getAllIngredientNames())
- `allRecipeTitles: {id: string; title: string}[]` (loaded once from getAllRecipeTitles())
- `query: string` (search bar text)
- `ingredientChips: string[]` (pinned ingredient chips)
- `suggestions: AutocompleteSuggestion[]` (filtered autocomplete list)
- `results: RecipeListItem[]` (search results)
- `recentViews: RecipeListItem[]` (shown when query='' and chips=[])
- `profile: Profile | null`
- `searchLoading: boolean`

On mount: load allIngredients + allRecipeTitles in parallel; load profile; load recent views (getRecentViews + join with recipe info); setProfileLoaded=true.

Autocomplete logic (runs on every query change via useMemo or useEffect):
```
If query.length >= 2:
  ingredientMatches = allIngredients.filter(n => n.toLowerCase().includes(query.toLowerCase())).slice(0, 8)
  recipeMatches = allRecipeTitles.filter(r => r.title.toLowerCase().includes(query.toLowerCase())).slice(0, 4)
  suggestions = [...ingredientMatches.map(v => ({value:v, type:'ingredient', recipeId:null})),
                 ...recipeMatches.map(r => ({value:r.title, type:'recipe', recipeId:r.id}))]
Else:
  suggestions = []
```

No DB call on every keypress — filter in JS only (Pitfall 7 prevention).

When user selects ingredient suggestion: call handleSelectIngredient(name) — adds to ingredientChips, clears query, hides suggestions.
When user selects recipe suggestion: `router.push('/recipe/' + recipeId)`.
When ingredientChips changes (or on mount if chips present): call searchRecipesByIngredients(ingredientChips, profile.allergens), set results.
When chips=[] and query='': show recent views using RecipeCardRow list.
When chips present: show results using FlashList with RecipeCardGrid (numColumns=2).

Layout:
- Top: TextInput search bar (placeholder="Malzeme veya tarif ara...", borderRadius=12, borderWidth=1, borderColor='#E5E7EB', padding=12, fontSize=15, backgroundColor='#F9FAFB')
- Below search bar: IngredientChips component (renders pinned ingredient chips; × calls handleRemoveChip)
- Autocomplete dropdown: absolute positioned FlatList below search bar, zIndex=100, backgroundColor='#FFFFFF', borderWidth=1, borderColor='#E5E7EB', borderRadius=12, maxHeight=280, shadow. Each suggestion row: text + small tag badge ('Malzeme' or 'Tarif'). Rendered only when suggestions.length > 0.
- Results area: RecipeCardGrid FlashList when chips active; RecipeCardRow list for recent views when idle.
- Empty results state: "Eşleşen tarif bulunamadı. Başka malzemeler dene." centered text.
