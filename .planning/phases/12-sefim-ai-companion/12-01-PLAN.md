    1. Create useSefim.ts with the following exports:

    **Types:**
    ```typescript
    export interface SefimMessage {
      role: 'user' | 'assistant';
      text: string;
      isChip?: boolean;  // true for pre-loaded Q&A responses
    }

    export interface SefimContext {
      recipeName: string;
      totalSteps: number;
      currentStepIndex: number;
      currentStepInstruction: string;
      currentStepWhy: string;
      currentStepCommonMistake: string;
      currentStepRecovery: string;
      userSkillLevel: string;
      ingredientSwaps: Record<string, string>;
      servingMultiplier: number;
      checkpoint: string | null;
      warning: string | null;
    }
    ```

    **Pure functions (exported for testing):**
    - `buildSefimContext(recipe, stepIndex, skillLevel, swaps, servings)` -- assembles minimal context for AI call
    - `getLingerThreshold(step)` -- returns ms: `step.timerSeconds ? step.timerSeconds * 1.5 * 1000 : 120_000`

    **Hook: `useSefim(recipe, currentStepIndex, skillLevel, swaps, servings)`**
    - State: `messages: SefimMessage[]`, `isLoading: boolean`, `lingerActive: boolean`, `isOpen: boolean`
    - `open()` / `close()` -- toggle sheet visibility
    - `handleChipTap(qa: SefimQA)` -- adds question + answer to messages instantly, no network
    - `handleOpenQuestion(text: string)` -- adds user message, sets isLoading, calls `supabase.functions.invoke('sefim-ask', { body: { question, context: buildSefimContext(...) } })`, adds assistant response, clears isLoading. On error, adds fallback message "Baglanti sorunu, tekrar dene".
    - Linger detection: useEffect on currentStepIndex that sets a setTimeout for getLingerThreshold. Sets lingerActive=true when fired. Cleanup clears timeout. Resets lingerActive to false on step change.
    - `clearMessages()` -- resets messages array (called on close or step change, up to consumer)

    2. Create supabase/functions/sefim-ask/index.ts (Deno Edge Function):
    - Reads `OPENAI_API_KEY` from `Deno.env.get()`
    - Receives `{ question, context }` from request body
    - Builds system prompt in Turkish (see RESEARCH.md Pattern 4):
      - Role: "Sen Sef'im adinda bir asci yardimcisisin"
      - Context: recipe name, current step, skill level
      - Constraint: only answer cooking-related questions; redirect off-topic with friendly Turkish message
      - Style: short, friendly, professional answers
    - Calls OpenAI chat completions API (gpt-4o-mini, max_tokens: 300, temperature: 0.7)
    - Returns `{ answer: string }`
    - Error handling: returns `{ answer: "Bir sorun olustu, tekrar dene" }` on failure
