# Feature: Navigation

## The Four Tabs

| Tab | Job |
|-----|-----|
| Feed | Passive discovery — shown something good when he doesn't know what he wants |
| Search | Active discovery — goes here when he has something specific in mind |
| Cookbook | Personal space — recipes to make and recipes already made |
| Profile | Identity in the app — preferences that shape everything he sees |

## Visibility Rule

Nav bar visible on every screen except Cooking Mode, where it disappears entirely. During Cooking Mode the only exit is the explicit exit button in the step screen header, which triggers a confirmation modal. Intentional — accidental nav tap mid-cook would end session with no recovery.
